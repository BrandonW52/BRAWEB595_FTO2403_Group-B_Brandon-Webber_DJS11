## Run Locally

Clone the project

```bash
  git clone https://github.com/BrandonW52/BRAWEB595_FTO2403_Group-B_Brandon-Webber_DJS11.git
```

Go to the project directory

```bash
  cd BRAWEB595_FTO2403_Group-B_Brandon-Webber_DJS11
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
